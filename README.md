# Zero Assist

AI-powered assistant with Google Calendar integration.

## Features

- Chat with AI assistant
- Voice input with audio transcription (OpenAI Whisper)
- View, create, update, and delete calendar events
- Search for events
- Add reminders and invite guests

## Setup

See [CLAUDE.md](./CLAUDE.md) for detailed setup instructions.

## Development

```bash
pnpm install
cp .env.example .env  # then fill in your credentials
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)
